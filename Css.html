CSS - Cascading Style Sheets

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de documentos HTML. Com o CSS, você pode controlar o layout, as cores, as fontes e outros aspectos visuais das páginas web.

Conceitos Básicos do CSS


  • Seletores: Permitem selecionar elementos HTML para aplicar estilos.
  • Propriedades: Definem os aspectos visuais que você deseja alterar.
  • Valores: Especificam os valores das propriedades.

Como Incluir CSS em um Documento HTML


Existem três maneiras principais de incluir CSS em um documento HTML:

  1. CSS Externo: Usando a tag <link> no <head> do documento.
  2. CSS Interno: Usando a tag <style> dentro do <head>.
  3. CSS Inline: Usando o atributo "style" diretamente em um elemento HTML.

História do CSS


CSS foi desenvolvido por Håkon Wium Lie em 1994 enquanto trabalhava no CERN. A primeira especificação oficial, CSS1, foi publicada em 1996 pelo W3C.


Seletores


  • Tipo: p { color: blue; }
  • Classe: .highlight { background-color: yellow; }
  • ID: #header { font-size: 24px; }

Propriedades Comuns


  • color: color: red;
  • font-size: font-size: 16px;
  • background-color: background-color: lightgray;
  • margin: margin: 10px;
  • padding: padding: 5px;
  • border: border: 1px solid black;

Exemplo de CSS Externo


<link rel="stylesheet" href="styles.css">

Exemplo de CSS Interno


<style>
  body {
    font-family: Arial;
    background-color: #f0f0f0;
  }
  h1 {
    color: blue;
  }
  p {
    color: green;
  }
</style>

Exemplo de CSS Inline


<h1 style="color: blue;">Olá, Mundo!</h1>

Conclusão


O CSS é uma ferramenta poderosa para estilizar páginas web. Compreender os conceitos básicos é essencial para qualquer desenvolvedor web.
