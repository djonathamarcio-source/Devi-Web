O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web. Ela estrutura o conteúdo exibido nos navegadores e pode ser complementada por CSS e JavaScript para criar páginas modernas e interativas.

História da Internet


A Internet: Origem e Evolução


A internet surgiu na década de 1960 com a ARPANET, um projeto do Departamento de Defesa dos EUA. O objetivo era criar uma rede de comunicação segura e resiliente. Com o tempo, outras redes foram conectadas, formando a internet global que conhecemos hoje, essencial para comunicação, acesso à informação e serviços online.


Estrutura Básica do HTML


Um documento HTML é composto por elementos representados por tags. Cada tag tem uma função específica e pode conter atributos. A estrutura básica inclui:


  • <!DOCTYPE html>: Declara o tipo de documento.
  • <html>: Tag raiz do documento.
  • <head>: Metadados, título, links e scripts.
  • <body>: Conteúdo visível da página.



<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemplo HTML</title>
</head>
<body>
    <h1>Título da Página</h1>
    <p>Conteúdo da página.</p>
</body>
</html>




Principais Elementos HTML

  • <h1> a <h6>: Cabeçalhos de diferentes níveis.
  • <p>: Parágrafo de texto.
  • <a>: Link para outra página ou recurso (href).
  • <img>: Imagem (src e alt).
  • <ul> e <ol>: Listas não ordenadas e ordenadas.
  • <li>: Item de lista.
  • <div>: Contêiner genérico para agrupar elementos.
  • <span>: Contêiner em linha para estilizar partes do texto.
  • <form>: Formulário para entrada de dados.
  • <table>: Tabela para organizar dados.



Links e Navegação


Os links permitem a navegação entre páginas e recursos. Exemplo: 


<a href="https://www.exemplo.com">Visite nosso site</a>

Para navegação interna: 


<a href="pagina2.html">Ir para a Página 2</a>

Inputs e Formulários


Inputs permitem a entrada de dados pelo usuário. Exemplos:

  • Texto: <input type="text">
  • Senha: <input type="password">
  • Email: <input type="email">
  • Número: <input type="number">
  • Checkbox: <input type="checkbox">
  • Radio: <input type="radio">
  • Submit: <input type="submit">
  • Reset: <input type="reset">
  • Textarea: <textarea>



<form action="processar.php" method="post">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome" required>
    <label for="email">E-mail:</label>
    <input type="email" id="email" name="email" required>
    <input type="submit" value="Enviar">
</form>



Elementos Semânticos


Elementos semânticos ajudam a estruturar o conteúdo de forma lógica e acessível:


  • <header>: Cabeçalho da página ou seção.
  • <nav>: Navegação.
  • <main>: Conteúdo principal.
  • <article>: Conteúdo independente.
  • <section>: Seção de conteúdo.
  • <aside>: Conteúdo complementar.
  • <footer>: Rodapé.
  • <figure> e <figcaption>: Imagem e legenda.
  • <time>: Data ou hora.


Elementos de Bloco e Inline


Elementos de bloco ocupam toda a largura disponível e começam em nova linha (ex: <div>, <p>, <h1>).
Elementos inline ocupam apenas o espaço necessário (ex: <span>, <a>, <img>).



Tabelas e Formulários


Tabelas organizam dados em linhas e colunas: 



<table>
    <tr>
        <th>Nome</th>

        <th>Cidade</th>
    </tr>
    <tr>
        <td>João</td>
        <td>25</td>

    </tr>

Listas

Listas não ordenadas (<ul>): 


<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

Listas ordenadas (<ol>): 


<ol>
    <li>Primeiro Item</li>
    <li>Segundo Item</li>
</ol>


Mídias


HTML permite incorporar imagens, vídeos e áudios: 


<img src="imagem.jpg" alt="Descrição">
<video src="video.mp4" controls>Seu navegador não suporta vídeo.</video>
<audio src="audio.mp3" controls>Seu navegador não suporta áudio.</audio>


Responsividade


Para tornar o site responsivo, use a meta tag viewport e imagens flexíveis: 



<meta name="viewport" content="width=device-width, initial-scale=1.0">
<img src="imagem-pequena.jpg" 
     srcset="imagem-media.jpg 600w, imagem-grande.jpg 1200w" 
     sizes="(max-width: 600px) 100vw, (max-width: 1200px) 50vw, 33vw" 
     alt="Descrição da imagem">
