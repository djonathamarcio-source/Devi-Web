JavaScript


JavaScript é uma linguagem de programação que permite criar conteúdo dinâmico em páginas web. Com JavaScript, você pode adicionar interatividade, validar formulários, manipular elementos HTML e muito mais.


Exemplo Simples de JavaScript


<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemplo de JavaScript</title>
    <script>
        function mostrarMensagem() {
            alert("Olá, Mundo!");
        }
    </script>
</head>
<body>
    <h1>Exemplo de JavaScript</h1>
    <button onclick="mostrarMensagem()">Clique aqui</button>
</body>
</html>

História do JavaScript


JavaScript foi criado por Brendan Eich em 1995 enquanto trabalhava na Netscape Communications Corporation. Originalmente, a linguagem foi desenvolvida em apenas 10 dias e foi inicialmente chamada de Mocha, depois renomeada para LiveScript e, finalmente, para JavaScript. A escolha do nome "JavaScript" foi uma estratégia de marketing para capitalizar a popularidade da linguagem de programação Java na época, embora as duas linguagens sejam distintas e tenham propósitos diferentes.

JavaScript rapidamente se tornou uma das linguagens de programação mais populares para desenvolvimento web, permitindo que os desenvolvedores criassem páginas web interativas e dinâmicas. Com o tempo, a linguagem evoluiu significativamente, com a introdução de novas funcionalidades e padrões, como o ECMAScript, que padroniza a linguagem e garante compatibilidade entre diferentes navegadores.

Hoje, o JavaScript é amplamente utilizado não apenas no desenvolvimento web, mas também em servidores (com Node.js), aplicativos móveis, desenvolvimento de jogos e muito mais. A comunidade JavaScript é vibrante e ativa, contribuindo constantemente para o crescimento e aprimoramento da linguagem.

Conceitos Básicos de JavaScript


JavaScript é uma linguagem de programação amplamente utilizada para criar páginas web interativas. Aqui estão alguns conceitos básicos de JavaScript:


  • Variáveis: São usadas para armazenar dados. Em JavaScript, você pode declarar variáveis usando as palavras-chave var, let ou const. Exemplo: let nome = "João";
  • Tipos de Dados: JavaScript suporta vários tipos de dados, incluindo números, strings, booleanos, arrays, objetos e nulos. Exemplo: let idade = 25;
  • Funções: Blocos de código que podem ser reutilizados. Você pode definir uma função usando a palavra-chave function. Exemplo: 

    function saudacao() {
    console.log("Olá!");
}

  • Eventos: JavaScript pode responder a eventos do usuário, como cliques de botão ou movimentos do mouse. Exemplo: 

    <button onclick="saudacao()">Clique aqui</button>

  • Manipulação do DOM: JavaScript pode alterar o conteúdo e o estilo de elementos HTML dinamicamente. Exemplo: 

    document.getElementById("meuElemento").innerHTML = "Novo Conteúdo";

  • Condicionais: Permitem executar diferentes blocos de código com base em condições. Exemplo: 

    if (idade >= 18) {
    console.log("Você é maior de idade.");
} else {
    console.log("Você é menor de idade.");
}

  • Loops: Permitem repetir um bloco de código várias vezes. Exemplo: 

    for (let i = 0; i < 5; i++) {
    console.log(i);
}

  • Objetos: Estruturas que armazenam dados em pares chave-valor. Exemplo: 

    let pessoa = {
    nome: "Maria",
    idade: 30
};
