O que é Desenvolvimento Web?


Desenvolvimento web é o processo de criação e manutenção de sites e aplicações web. Envolve várias disciplinas, incluindo design, programação, banco de dados e otimização para mecanismos de busca (SEO).


Principais Tecnologias


  • HTML (HyperText Markup Language): A linguagem de marcação padrão para criar páginas web. Ela estrutura o conteúdo da web usando elementos como títulos, parágrafos, links e imagens.
  • CSS (Cascading Style Sheets): Uma linguagem de estilo usada para descrever a apresentação de um documento HTML. Com CSS, você pode controlar o layout, cores, fontes e outros aspectos visuais das páginas web.
  • JavaScript: Uma linguagem de programação que permite adicionar interatividade às páginas web. Com JavaScript, você pode criar efeitos dinâmicos, validar formulários, manipular elementos HTML e muito mais.

Exemplos de Código


Abaixo estão alguns exemplos simples de código para cada uma das principais tecnologias:


Exemplo de HTML



<!DOCTYPE html>
<html lang="pt-br">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemplo de HTML</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <p>Este é um exemplo simples de uma página HTML.</p>
</body>
</html>

Exemplo de CSS



    body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
    margin: 20px;
}
h1 {
    color: #007BFF;
}
p {
    font-size: 16px;
    line-height: 1.5;
}

Exemplo de JavaScript


<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemplo de JavaScript</title>
    <script>
        function mostrarMensagem() {
            alert("Olá, Mundo!");
        }
    </script>
</head>
<body>
    <h1>Exemplo de JavaScript</h1>
    <button onclick="mostrarMensagem()">Clique aqui</button>
</body>
</html>
